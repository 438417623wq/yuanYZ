# 隐私说明

本应用的目标是在安卓设备本地完成视频转文字、语音转文字和图片转文字，尽量减少用户音视频、图片和识别文本离开设备的风险。

## 本地处理

- 视频转文字：应用从用户选择的视频中提取音频，并在本地进行语音识别。
- 语音转文字：应用使用内置或本地模型进行识别。
- 图片转文字：应用使用设备端 OCR 能力识别图片文字。
- 转写结果：识别历史保存在应用本地存储中，用户可以在应用内查看、编辑或删除。

## 权限说明

- 麦克风权限：用于实时语音输入和录音转写。
- 文件/媒体访问：由 Android 系统文件选择器或媒体授权流程提供，用于读取用户主动选择的视频、音频或图片。
- 网络访问：当前项目声明了网络权限，主要用于兼容第三方 SDK、模型更新或系统服务行为。项目代码本身不主动把用户选择的音频、视频、图片或识别文本上传到项目作者服务器。

## 第三方 SDK 与可能的联网行为

本项目使用 Google ML Kit Text Recognition 进行 OCR。根据 Google ML Kit 条款与隐私说明，ML Kit API 对输入数据的处理在设备端完成，不会把输入数据和识别输出发送给 Google；但 ML Kit API 可能联系 Google 服务器获取修复、更新模型和硬件兼容信息，也可能发送 API 性能和使用指标。

本项目还集成了 Vosk、sherpa-onnx、SenseVoice / FunASR 等本地语音识别组件和模型。这些组件用于在设备上执行本地识别。请以各组件和模型的上游许可证、隐私说明和实际版本行为为准。

## 本地保存与备份

应用会在本地保存转写记录，包括文件标题、来源 URI、识别文本、分段信息、状态和创建时间。

为降低识别历史进入云端备份的风险，项目已关闭 Android 自动备份。用户卸载应用、清除应用数据或更换设备时，本地记录可能被删除。

## 不收集的内容

项目作者没有搭建用于接收用户音频、视频、图片、识别文本或转写历史的服务器。除第三方 SDK 或系统服务可能产生的必要通信外，项目代码不主动上传用户内容。

## 用户建议

- 处理敏感音视频或图片前，请确认设备环境可信。
- 商业发布或上架应用市场前，请根据实际 APK、依赖版本和目标地区法律完善隐私政策。
- 如果后续新增账号、云同步、广告、统计、崩溃上报或远程模型下载功能，应同步更新本隐私说明。
