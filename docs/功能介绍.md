# 本地全能转文字功能介绍

更新时间：2026-05-12

## 应用定位

本地全能转文字是一款面向安卓手机的本地转文字工具，核心功能是把视频、语音和图片内容转换成文字，也支持从视频中单独导出音频。项目目标是尽量在设备本地完成识别、校对、播放、音频导出和字幕翻译流程，减少用户音频、视频、图片和识别文本上传到云端带来的隐私风险。

应用名称：本地全能转文字  
安装包名：com.yuanYZ  
最低系统：Android 7.0，API 24

## 功能总览

| 功能 | 说明 |
| --- | --- |
| 视频转文字 | 选择本地视频后提取音频，使用本地语音识别模型生成文字和字幕片段。 |
| 语音转文字 | 支持导入音频文件，也支持实时录音识别。 |
| 视频转音频 | 从本地视频中导出 M4A 或 WAV 音频文件，导出任务可在后台继续。 |
| 图片转文字 | 支持图片 OCR，把截图、照片、长图中的文字识别出来。 |
| 后台转换 | 视频、音频、图片识别和视频转音频导出可在切到后台后继续运行，并通过通知显示进度。 |
| 性能模式 | 提供省内存、均衡、高性能三种模式，适配不同手机性能和内存情况。 |
| 历史记录 | 自动保存每次转换结果，可按全部、图片、语音、视频、视频音频筛选。 |
| 结果操作 | 支持复制、分享、查看全文、删除历史记录。 |
| 字幕校对 | 视频转换完成后可进入字幕校对界面，播放视频并逐段修改字幕。 |
| 播放器能力 | 支持播放、暂停、进度拖动、快退快进、倍速、横竖屏、全屏播放。 |
| 历史视频切换 | 全屏播放器右上角可打开已转换视频列表，快速切换历史视频。 |
| 字幕翻译 | fullOffline 版支持本地 ONNX 翻译模型，将字幕翻译为其它语言。 |
| 关于与声明 | 右上角关于按钮展示作者信息、第三方组件和模型声明。 |

## 转换首页

首页提供视频转文字、语音转文字、视频转音频、图片转文字四类入口。用户选择文件或开始录音后，应用会显示当前转换状态、进度和最新结果。

首页还包含本地模型状态和性能模式设置：

- 精准语音模型已内置时，会提示可以直接开始识别。
- 如果模型缺失，可以使用备用导入功能导入本地语音模型压缩包。
- 性能模式会影响语音分块长度、识别线程数、图片缩放尺寸和长图切片高度。

## 视频转文字

视频转文字用于把本地视频里的语音内容转为文字，并生成可校对的字幕片段。

主要流程：

1. 点击视频转文字。
2. 选择识别语言。
3. 选择手机本地视频文件。
4. 应用在后台提取视频音频。
5. 使用本地语音识别模型转写。
6. 自动保存结果到历史记录。

视频转换结果包含：

- 完整识别文本。
- 分段字幕数据。
- 原视频路径。
- 转换状态和创建时间。

如果转换完成后进入历史记录，可以点击视频记录里的“编辑字幕”进入字幕校对界面。

## 视频转音频

视频转音频用于从本地视频中单独导出音频文件。

主要流程：

1. 点击视频转音频。
2. 选择本地视频文件。
3. 选择导出格式。
4. 选择手机里的保存位置和文件名。
5. 应用在后台导出音频，并把导出记录保存到历史记录。

支持格式：

- M4A/AAC：速度快、体积小，会直接封装视频里的 AAC 音轨；如果视频音轨不是 AAC，需要选择 WAV。
- WAV：兼容性更强，会把视频音轨解码为 WAV 文件，适合后续转写或其它工具处理。

导出完成后，历史记录里的“视频音频”记录支持分享音频文件。

## 语音转文字

语音转文字支持两种来源：

- 实时录音：直接使用手机麦克风识别。
- 导入音频：选择本地音频文件后转写。

实时录音能力：

- 开始录音。
- 暂停录音。
- 继续录音。
- 保存识别内容到历史记录，并同时保留本次录音 WAV 文件。
- 清空当前草稿。
- 连续录音开关：使用本地麦克风持续分段识别，适合较长时间口述。
- 实时降噪开关：设备支持时启用系统降噪、自动增益和回声消除，并对低幅噪声做轻量抑制。

实时录音已改为 `AudioRecord + sherpa-onnx / SenseVoice` 本地识别，不再依赖 Android 系统语音识别服务。这样能减少部分手机点击开始录音无反应、系统语音服务不可用或网络识别异常的问题。

保存实时录音时，历史记录会同时保存识别文字和应用私有目录中的 WAV 音频；导入音频时，历史记录会保留原音频文件入口和识别文字。历史记录中可使用“分享音频”把对应音频发送到其它应用。

导入音频时，应用会把音频分块处理，降低一次性读取大文件导致的内存压力。

## 图片转文字

图片转文字用于识别相册、截图、文档照片和长图中的文字。

主要能力：

- 选择本地图片后自动识别文字。
- 自动保存 OCR 结果到历史记录。
- 根据性能模式缩放图片，减少内存占用。
- 长图会分块切片识别，降低内存溢出风险。

图片 OCR 使用 Google ML Kit Text Recognition。识别输入和输出以设备端处理为目标，但 ML Kit 可能根据其条款进行模型更新、兼容性信息或性能指标通信，正式发布时应在隐私说明中保留相关说明。

## 识别语言

语音和视频转文字支持以下识别语言：

- 普通话
- 英文
- 日语
- 韩语
- 粤语
- 其它方言

“其它方言”用于四川话、上海话、口音普通话、中英混合等更复杂场景。

## 性能模式

应用提供三种性能模式：

| 模式 | 适用场景 |
| --- | --- |
| 省内存 | 低内存手机优先，速度慢一点，但更稳定。 |
| 均衡 | 默认推荐，在稳定性和速度之间折中。 |
| 高性能 | 新手机优先，速度更快，但更占用内存。 |

性能模式会影响：

- 语音识别线程数。
- 音频分块秒数。
- 图片最大边长。
- 长图 OCR 切片高度。

## 后台转换

视频、音频和图片转换已接入后台服务。开始转换后，即使用户切到后台，转换也会继续进行。

后台转换特点：

- 使用前台服务通知显示转换状态。
- 通知中显示进度。
- 支持取消后台转换。
- 使用 Wake Lock 降低息屏后任务被系统暂停的概率。
- 转换完成后结果自动写入历史记录。

## 历史记录

历史记录会自动保存每次转换结果。

支持的操作：

- 查看全部历史。
- 按图片、语音、视频、视频音频筛选。
- 复制识别文本。
- 分享识别文本。
- 分享语音记录或视频转音频记录对应的音频文件。
- 删除单条记录。
- 清空全部历史记录。
- 视频记录可进入字幕校对。

历史记录中会显示转换类型、状态、创建时间和文本预览。

## 字幕校对与视频播放器

视频转换成功后，可以进入字幕校对界面。该界面把播放器和字幕列表放在一起，方便边看视频边修正文字。

播放器能力：

- 播放和暂停。
- 拖动进度条。
- 快退 10 秒。
- 快进 10 秒。
- 倍速播放，支持 0.5x、0.75x、1x、1.25x、1.5x、2x。
- 自动、竖屏、横屏三种方向模式。
- 全屏播放。
- 显示或隐藏字幕。
- 原文、译文、双语字幕显示模式。

字幕编辑能力：

- 点击单条字幕进行文字修改。
- 删除字幕片段。
- 批量提前 0.5 秒。
- 批量延后 0.5 秒。
- 保存修改后的字幕。
- 复制完整字幕文本。
- 分享完整字幕文本。

如果原视频文件无法读取，可以在字幕校对界面重新选择视频文件。

## 全屏历史视频切换

全屏播放器右上角提供历史视频按钮，可打开已转换视频列表。

该功能用于：

- 查看已经转换过的视频记录。
- 显示视频缩略图、标题和字幕段数。
- 快速切换到其它已转换视频。
- 当前字幕有未保存修改时，提示保存并切换、不保存切换或取消。

## 本地字幕翻译

fullOffline 版内置 OPUS-MT / Marian ONNX 量化翻译模型，并通过 ONNX Runtime Android 在本地执行字幕翻译。

翻译入口：

1. 打开视频字幕校对界面。
2. 点击“翻译字幕”。
3. 选择原语言和目标语言。
4. 选择显示方式。
5. 必要时点击“启动模型”预热当前翻译路线。
6. 点击“开始翻译”。

显示方式：

- 只看译文。
- 双语字幕。

当前已内置翻译方向：

- 英文 -> 中文
- 中文 -> 英文
- 英文 -> 日文
- 日文 -> 英文
- 韩文 -> 英文

当前通过中转支持：

- 中文 -> 日文，通过英文中转。
- 日文 -> 中文，通过英文中转。
- 韩文 -> 中文，通过英文中转。

当前限制：

- 暂未内置英文 -> 韩文模型，因此韩文暂不作为目标语言开放。
- 标准版不包含本地字幕翻译模型，需要使用 fullOffline 版。
- 翻译质量取决于内置模型能力，适合字幕辅助理解，正式发布字幕建议人工复核。

## 隐私说明

项目代码不主动把用户选择的音频、视频、图片或识别文本上传到项目作者服务器。

需要注意：

- 视频和音频识别以本地模型处理为主。
- 字幕翻译在 fullOffline 版中使用内置 ONNX 模型本地推理。
- 图片 OCR 使用 ML Kit，需按其条款说明可能的模型更新、兼容性信息或指标通信。
- APP 不开启云端账号体系，不依赖项目作者服务器保存用户内容。

更完整的隐私说明见 `PRIVACY.md`。

## 版本说明

当前项目包含两个构建版本：

| 版本 | 说明 |
| --- | --- |
| standard | 标准版，包含视频、语音、图片转文字、视频转音频、历史记录、字幕校对和播放器等核心能力。 |
| fullOffline | 全离线翻译版，在标准版基础上内置本地字幕翻译模型。 |

fullOffline 版体积较大，首次使用翻译时会把模型文件从 APK 资源复制到应用私有目录，启动模型可能需要等待一段时间。
